It is important to learn how to pick the right tools for the job. blog: https://www.urban.org/research/publication/do-no-harm-guide-centering-accessibility-data-visualization venue_url: https://www.urban.org/research/publication/do-no-harm-guide-centering-accessibility-data-visualization year: 2022 --- As the data visualization field continues to expand and grow, accessibility considerations need to be central to content creators’ analyses and communications. If creators continue to use the same methods and tools without interrogation, the current state of inaccessibility will only worsen. To make the future more accessible, creators first need to recognize what tools and solutions aren’t a good fit, become more knowledgeable about accessibility, explore how we can improve the tools we have, and build new tools entirely.
